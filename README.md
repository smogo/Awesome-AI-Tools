## üí¨ Chat-based LLMs (no setup ‚Äì just type a prompt)

Open in your browser and start chatting immediately ‚Äì no API keys or install required.

- **Microsoft Copilot**  
  https://copilot.microsoft.com/  
  General-purpose AI assistant from Microsoft. Great for everyday Q&A, writing, brainstorming and browsing with built-in web access. Also powers Microsoft 365 Copilot Chat for work scenarios.  
  _Best for:_ people already in the Microsoft ecosystem, quick productivity & office workflows.  

- **OpenAI ‚Äì ChatGPT**  
  https://chatgpt.com/  
  Chat interface for OpenAI‚Äôs latest flagship models (e.g. GPT-4.x / GPT-5.x, depending on plan). Handles text, code, images and voice, with strong reasoning and coding abilities.  
  _Best for:_ general use, coding help, learning, creative writing, multimodal tasks.  

- **Google ‚Äì Gemini**  
  https://gemini.google.com/  
  Google‚Äôs AI assistant built on the Gemini model family. Helps with writing, planning, search and multimodal tasks; deeply integrated with Google ecosystem (Drive, Docs, Gmail, etc.).  
  _Best for:_ Google users (Gmail, Docs, Android), long-form tasks & search.  

- **Anthropic ‚Äì Claude**  
  https://claude.ai/  
  Chat interface for the Claude 3.x / 3.5 family ‚Äì known for long context, careful reasoning and working with big documents. Official apps are available on web and mobile.  
  _Best for:_ analysis of long docs, structured writing, safer / more conservative assistant.  

- **Meta ‚Äì Meta AI (Llama-based)**  
  https://www.meta.ai/  
  Assistant built on Meta‚Äôs latest Llama models, available on the web and inside Meta products (Instagram, Facebook, WhatsApp in many regions). Can chat, answer questions and generate images.  
  _Best for:_ quick consumer chat & image generation, especially if you use Meta apps.  

- **Mistral ‚Äì Le Chat**  
  https://chat.mistral.ai/  
  Web chat for Mistral‚Äôs models (including Pixtral for multimodal). Strong European-centric option with focus on privacy and enterprise use.  
  _Best for:_ EU-friendly assistant, research, coding and image-capable chat.  

- **Alibaba ‚Äì Qwen Chat**  
  https://chat.qwen.ai/  
  Chat interface for Qwen models from Alibaba (text + vision + docs). Offers image/video understanding, generation, web search and document processing.  
  _Best for:_ users interested in the Chinese ecosystem, multimodal & office assistant.  

- **Perplexity ‚Äì AI Answer Engine**  
  https://www.perplexity.ai/  
  Search-style chat that combines multiple top models with live web browsing and citations. Positions itself as an ‚Äúanswer engine‚Äù, not a classic chatbot.  
  _Best for:_ research, fact-checking, quick answers with links to sources.  

- **Cohere ‚Äì Playground (Command / Aya models)**  
  https://dashboard.cohere.com/playground  
  Browser playground to test Cohere‚Äôs chat and embeddings models without writing code. Good way to explore enterprise-oriented models (Command, Aya).  
  _Best for:_ experimenting with Cohere models before integrating the API.  

- **PLLuM ‚Äì Polish Large Language Model**  
  https://pllum.clarin-pl.eu/pllum_8x7b

- **Bielik ‚Äì Polish Large Language Model**  
  https://chat.bielik.ai/  
  Bielik is a non-profit, community-based project dedicated to creating Polish language models (LLMs). The models understand the Polish language, culture, and context, operate locally, and are developed by the SpeakLeash Foundation, partners, and the community.
